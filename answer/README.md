# install & run

1. 安装node.js
2. 【可选】在浏览器中安装livereload插件：http://feedback.livereload.com/knowledgebase/articles/86242-how-do-i-install-and-use-the-browser-extensions-
3. npm install
4. grunt watch
5. 用浏览器打开http://localhost:8000/
